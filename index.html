✅ Sumber release resmi

Tombol download diarahkan ke asset GitHub Release, bukan ke link acak. Tombol Lihat GitHub Release disediakan agar user bisa memastikan versi dan file berasal dari repository yang benar.

🔐 Verifikasi SHA-256

Checksum SHA-256 membantu memastikan file yang diunduh sama dengan file rilis. Jika hasil checksum berbeda, jangan jalankan file tersebut dan download ulang dari release resmi.

🧩 Tidak menyimpan data user

Halaman download ini hanya membaca metadata latest.json. Tidak ada form login, tidak ada upload file, dan tidak ada pengumpulan data pribadi dari user.

🪟 Windows SmartScreen

Installer baru yang belum ditandatangani sertifikat resmi bisa memunculkan peringatan SmartScreen. Itu bukan selalu berarti virus, tetapi user tetap wajib mengecek sumber file dan checksum.

📦 Pilihan ZIP portable

Jika user ragu menjalankan installer EXE, tersedia versi ZIP portable/manual. ZIP memudahkan pengecekan isi file terlebih dahulu sebelum aplikasi dijalankan.

🚫 Tidak perlu mematikan antivirus

User tidak disarankan menonaktifkan Windows Defender atau antivirus. Jika file dianggap mencurigakan, lakukan scan ulang, cocokkan SHA-256, atau download ulang dari GitHub Release.

Cara cek SHA-256 di Windows PowerShell Get-FileHash .\MirrorSync-Setup-v1.3.1.exe -Algorithm SHA256
Catatan: SHA-256 bukan pengganti antivirus, tetapi berguna untuk membuktikan integritas file. File aman seharusnya berasal dari release resmi, checksum cocok, dan tidak meminta izin yang tidak berhubungan dengan fungsi aplikasi.
