Persiapan Windows

Install sekali, buka seperti aplikasi biasa.

Gunakan installer resmi, lalu selalu buka MirrorSync dari shortcut Desktop atau Start Menu.

Beginner friendlySecurity note

Cara install MirrorSync

  1. Download installer EXE resmi.
  2. Jalankan installer dan tunggu hingga selesai.
  3. Buka MirrorSync melalui shortcut Desktop.
Jika Windows SmartScreen muncul

Pastikan file berasal dari GitHub Release resmi dan cocokkan SHA-256. Klik More info lalu Run anyway hanya jika sumber dan checksum benar. Jangan menjalankan file internal dari folder aplikasi.

Tips singkatSimpan shortcut Desktop; itu jalur startup yang benar dan paling bersih.

Aktifkan Developer Options.

Koneksi USB

Paling aman untuk koneksi pertama.

USB memudahkan proses izin awal dan membantu memastikan HP dikenali sebelum beralih ke Wireless Debugging.

RecommendedBeginner friendly

Aktifkan USB Debugging

  1. Aktifkan USB Debugging di Developer Options.
  2. Sambungkan kabel data dan pilih File transfer jika diminta.
  3. Centang Always allow, lalu tekan Allow.
  4. Klik Refresh Devices, pilih HP, lalu Start Mirror.
HP hanya mengisi daya atau popup tidak muncul?

Ganti dengan kabel data. Jika popup izin tetap tidak muncul, pilih Revoke USB debugging authorizations lalu sambungkan ulang.

Tips singkatPastikan layar HP terbuka ketika menunggu popup izin.

Wireless Debugging / WiFi ADB

Mirror tanpa kabel, langkah demi langkah.

Wireless Debugging tersedia pada Android 11+. HP dan laptop/PC harus berada di WiFi yang sama selama pairing dan penggunaan.

Tanpa kabelNative retry in v1.3.11
i
Sebelum mulai

Aktifkan Wireless Debugging, jangan matikan WiFi saat mirror, dan ingat bahwa beberapa HP mengganti pairing code setiap kali menu pairing dibuka ulang.

ATahap pertama

Pair dengan pairing code

  1. Tap menu Wireless Debugging.
  2. Pilih Pair device with pairing code.
  3. Catat IP address, pairing port, dan pairing code enam digit.
  4. Masukkan ketiganya ke bagian Pair di MirrorSync.
  5. Klik Pair Device dan tunggu pesan sukses.
BSetelah pairing berhasil

Connect ke alamat utama

  1. Kembali ke halaman utama Wireless Debugging.
  2. Lihat bagian IP address & port utama.
  3. Masukkan IP dan connect port ke bagian Connect di MirrorSync.
  4. Klik Connect WiFi, lalu Refresh Devices.
  5. Pilih HP dan klik Start Mirror.
Jangan tertukar: Pairing Port bukan Connect Port.

Pairing code port digunakan satu kali untuk Pair. IP:Port utama Wireless Debugging digunakan untuk Connect. Angka port keduanya biasanya berbeda.

Untuk Pair192.168.1.10:37123 + codeUntuk Connect192.168.1.10:42111

Alur Wireless dalam lima langkah

Pair biasanya cukup sekali; Connect dapat memakai port baru setelah WiFi atau Wireless Debugging berubah.

01Aktifkan Wireless
02Pair Code
03Connect IP:Port
04Refresh
05Start Mirror
Tips singkatJika port berubah, gunakan angka terbaru yang tampil di halaman Wireless Debugging.
Saat koneksi terputus

Window Android tetap terbuka.

WiFi berubah, HP sleep, debugging mati, atau jaringan putus dapat membuat status offline. MirrorSync menjaga frame terakhir agar pekerjaan Anda tidak hilang mendadak.

New in v1.3.11

Retry Connection

Overlay muncul di atas frame terakhir. Aktifkan kembali Wireless Debugging, lalu tunggu auto reconnect atau klik Retry Sekarang.

Langkah pemulihan lengkap
  • Pastikan HP dan laptop/PC masih satu WiFi.
  • Matikan VPN atau hotspot isolation jika perangkat tidak saling terlihat.
  • Aktifkan kembali Wireless Debugging.
  • Jika gagal, ulangi Connect menggunakan IP:Port terbaru.
  • Jika pairing hilang, lakukan Pair ulang.
Retry SekarangTutup Mirror
Tips singkatTunggu beberapa detik setelah Wireless Debugging aktif sebelum menekan Retry.
Keyboard, autotext, dan password

Ketik dari PC dengan mode yang tepat.

Sebagian besar input bekerja langsung. Aplikasi password atau banking tertentu dapat membatasi keyboard virtual, paste, atau autotext demi keamanan.

Improved in v1.3.11

Jika keyboard belum masuk

  1. Klik area mirror agar fokus masuk.
  2. Klik Focus Mirror jika keyboard belum terbaca.
  3. Auto untuk umum, SDK untuk kompatibilitas.
  4. UHID untuk numpad/password jika HP mendukung.
  5. Gunakan Kirim Teks ke HP untuk teks panjang atau autotext.
Catatan autotext dan aplikasi banking

Autotext Windows bisa digunakan, tetapi jangan simpan password dalam snippet tidak aman. Beberapa aplikasi banking sengaja menolak paste atau autotext.

Tips singkatJika numpad/password tidak terbaca, coba UHID lalu klik Focus Mirror.

Troubleshooting

Status HP memberi petunjuk.

Ikuti tindakan berdasarkan status yang terlihat di dashboard.

Unauthorized

Unlock HP dan terima popup Allow USB debugging. Centang Always allow agar izin tidak terus ditanyakan.

Offline

Cabut-colok USB atau aktifkan kembali Wireless Debugging. Klik Restart ADB atau ulangi Connect IP:Port.

Tidak terdeteksi

Pastikan kabel mendukung data, HP dan PC satu WiFi, VPN tidak mengisolasi jaringan, lalu Refresh Devices.

FAQ

Jawaban untuk pertanyaan umum.

Buka pertanyaan yang sesuai dengan kondisi Anda.

Apa beda USB Debugging dan Wireless Debugging?

USB Debugging memakai kabel data dan paling stabil untuk setup awal. Wireless Debugging memakai WiFi lokal tanpa kabel dan tersedia pada Android 11+.

Kenapa Pairing Port dan Connect Port berbeda?

Pairing Port digunakan bersama pairing code untuk membangun kepercayaan. Connect Port adalah alamat layanan ADB utama setelah pairing. Gunakan masing-masing pada form yang benar.

Kenapa HP menjadi offline?

WiFi berubah, HP sleep, Wireless Debugging mati, VPN, atau jaringan terputus dapat membuat ADB offline. Aktifkan debugging kembali dan ulangi Connect jika perlu.

Apa itu Retry Connection?

Overlay yang muncul saat koneksi mirror putus. MirrorSync mempertahankan frame terakhir sambil mencoba reconnect otomatis atau menunggu Anda menekan Retry Sekarang.

Apakah window Android tertutup saat koneksi putus?

Pada native shell v1.3.11, window dipertahankan dan frame terakhir tetap terlihat selama proses reconnect.

Kenapa keyboard tidak terbaca?

Klik layar mirror atau Focus Mirror. Mulai dengan mode Auto, lalu coba SDK atau UHID jika aplikasi atau HP memerlukan metode input berbeda.

Kapan sebaiknya memakai UHID?

Gunakan UHID untuk pengalaman keyboard fisik, numpad, atau field password jika device mendukungnya. Kembali ke Auto bila ada masalah kompatibilitas.

Apakah bisa memakai autotext atau password?

Bisa melalui keyboard atau Kirim Teks ke HP, tetapi beberapa aplikasi banking sengaja memblokir paste/autotext. Jangan simpan password di snippet yang tidak aman.

Apakah clipboard saya disimpan?

Tidak. Clipboard hanya diproses saat diminta dan tidak masuk log, file, diagnostics, localStorage, atau history aplikasi.

Kenapa teks tidak bisa dicopy dari aplikasi tertentu?

Beberapa aplikasi Android, terutama aplikasi banking, memblokir copy/paste demi keamanan.

Apakah Auto Sync harus selalu aktif?

Tidak. Default Auto Sync adalah OFF. Aktifkan hanya saat dibutuhkan dan hanya ketika status mirror CONNECTED.

Apakah bisa copy OTP atau password?

Bisa jika aplikasi mengizinkan, tetapi gunakan dengan hati-hati dan klik Clear Clipboard Lokal setelah selesai.

Apakah terminal hitam sudah diperbaiki?

Ya, versi terbaru menggunakan startup tersembunyi dan penanganan console scrcpy agar terminal hitam tidak mengganggu penggunaan normal.

Apakah perlu install ADB atau scrcpy terpisah?

Tidak. MirrorSync menyertakan bundled Node runtime, ADB platform-tools, dan scrcpy.

Apakah data layar dikirim ke cloud?

Tidak. Dashboard dan streaming berjalan lokal di laptop/PC. Internet hanya diperlukan untuk download atau memeriksa update.
