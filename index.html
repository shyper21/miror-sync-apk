Kontrol Android tanpa mengganggu alur kerja Anda.

Semua fungsi penting berada dalam satu control center yang ringkas, dengan performa scrcpy dan koneksi ADB yang sudah terbukti.

Mirror cepat dan responsif

Pilih kualitas low, medium, atau high. MirrorSync menjalankan scrcpy secara lokal untuk tampilan tajam dan kontrol dengan latensi rendah.

Wireless yang praktis

Pair dan connect Android 11+ melalui Wireless Debugging tanpa harus terus memasang kabel USB.

Alias yang mudah dikenali

Beri inisial atau nama pada setiap perangkat agar banyak HP tetap mudah dibedakan dari dashboard.

Reconnect dan update terkontrol

Status koneksi dipantau dari dashboard. Saat versi baru tersedia, MirrorSync memberi indikator sehingga pengguna tetap menentukan kapan update diunduh dan dipasang.

Control center lengkap

Start/Stop Mirror, Wake Up, Always on top, Reset position, renderer fallback, dan diagnostics berada dalam dashboard lokal yang sama.

USB dan multi-device

Gunakan USB untuk koneksi paling stabil dan pilih beberapa HP yang terdeteksi sesuai inisial perangkat.

Auto update

Cek versi baru dari aplikasi, unduh, lalu pasang tanpa perlu menghapus instalasi lama secara manual.

Clean startup tanpa terminal hitam

Launcher dan proses backend berjalan tersembunyi agar MirrorSync terasa seperti aplikasi Windows yang bersih dan profesional.

Cara menghubungkan HP ke MirrorSync.

Ikuti urutan ini sekali saja. Setelah debugging diizinkan, koneksi berikutnya akan jauh lebih cepat.

01

Install MirrorSync di Windows

Download installer terbaru, jalankan sampai selesai, lalu buka MirrorSync dari shortcut Desktop. Jangan menjalankan file internal secara manual dari folder release.

Download installer
Visual Pengaturan Android
Langkah 02A · Settings

Buka Pengaturan Android

Masuk ke aplikasi Settings / Pengaturan pada HP.

  • Cari menu Tentang Ponsel atau About Phone.
  • Nama menu dapat berbeda pada Samsung, Xiaomi, OPPO, atau vivo.
Visual Developer Options
Langkah 02B · Developer

Aktifkan Developer Options

Tap Build Number / MIUI Version sebanyak 7 kali.

  • Masukkan PIN HP jika diminta.
  • Tunggu pesan bahwa mode developer sudah aktif.
Visual USB Debugging
Langkah 03 · Debugging

Aktifkan USB Debugging

Buka Developer Options lalu aktifkan USB Debugging.

  • Setujui peringatan keamanan Android.
  • Biarkan aktif selama HP digunakan dengan MirrorSync.
Visual izin debugging
Langkah 04A · Authorization

Izinkan komputer ini

Sambungkan kabel data, lalu pilih Allow saat popup debugging muncul.

  • Centang Always allow from this computer.
  • Gunakan kabel data, bukan kabel charging-only.
Visual perangkat terdeteksi
Langkah 04B · Start Mirror

Pilih device dan mulai mirror

Klik Refresh Devices, pilih HP yang muncul, lalu klik Start Mirror.

  • Gunakan inisial agar perangkat mudah dikenali.
  • USB adalah pilihan terbaik untuk koneksi pertama.
Visual WiFi ADB
Langkah 05 · Wireless

Gunakan WiFi ADB tanpa kabel

Pastikan HP dan laptop berada di jaringan WiFi yang sama.

  • Aktifkan Wireless Debugging di Developer Options.
  • Masukkan pairing IP, port, dan code ke MirrorSync.
  • Jika gagal, ulangi pairing atau sambungkan USB lebih dahulu.
Langkah 06

HP belum terdeteksi?

Coba pemeriksaan singkat ini dari atas ke bawah. Biasanya masalah selesai tanpa instal ulang.

  • Pastikan USB Debugging aktif.
  • Ganti kabel jika hanya bisa charging.
  • Klik Refresh Devices.
  • Cabut lalu colok ulang USB.
  • Restart ADB dari MirrorSync.
  • Izinkan popup debugging pada HP.
  • Pastikan WiFi HP dan laptop sama.
  • Restart MirrorSync jika diperlukan.
Pilih sesuai kebutuhan

Dua mode koneksi, satu pengalaman.

Mulai dengan USB untuk konfigurasi pertama, lalu pindah ke WiFi jika ingin bekerja tanpa kabel.

USB Mode

Paling stabil, latensi konsisten, dan paling mudah untuk otorisasi pertama. Pastikan kabel mendukung transfer data.

FAQ MirrorSync.

Hal penting yang paling sering ditanyakan sebelum memasang dan menghubungkan perangkat.

Apakah MirrorSync perlu internet?

Tidak untuk mirroring lokal. Internet hanya diperlukan saat mengunduh installer atau memeriksa update.

Apakah bisa digunakan tanpa kabel?

Bisa pada Android 11+ melalui Wireless Debugging, selama HP dan laptop berada di WiFi yang sama.

Kenapa HP tidak muncul?

Periksa USB Debugging, kabel data, popup izin pada HP, lalu klik Refresh Devices atau restart ADB.

Kenapa Windows menampilkan warning?

Installer belum code-signed. Unduh hanya dari GitHub Release resmi dan cocokkan SHA-256 yang ditampilkan.

Apakah layar HP dikirim ke server?

Tidak. ADB, scrcpy, dashboard, dan stream layar berjalan lokal pada laptop.

Apakah harus install ADB terpisah?

Tidak. Paket MirrorSync menyertakan runtime, platform-tools ADB, dan scrcpy yang diperlukan.

Kenapa versi lama menampilkan terminal hitam?

Proses launcher dan scrcpy sebelumnya masih dapat membuka console host pada kondisi tertentu, terutama setelah restart.

Apakah terminal hitam sudah diperbaiki?

Ya. MirrorSync v1.3.10 membawa clean startup dan penanganan console scrcpy yang lebih aman.
